## Στόχος
Το έργο **eproipologismos** επιδιώκει να καταστήσει τον ελληνικό κρατικό προϋπολογισμό **διαφανή, κατανοητό και διαδραστικό**.  
Η εφαρμογή προσφέρει ένα καλαίσθητο γραφικό περιβάλλον, όπου ο χρήστης (πολίτης ή κρατικός φορέας) μπορεί να:

- Εξετάζει έσοδα και δαπάνες ανά κατηγορία και έτος  
- Προσομοιώνει σενάρια αλλαγών (π.χ. αύξηση φόρων, μείωση δαπανών)  
- Συγκρίνει διαφορετικά έτη ή σενάρια  
- Βλέπει οπτικοποιημένα δεδομένα μέσω γραφημάτων

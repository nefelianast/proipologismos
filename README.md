# Προϋπολογισμός 

## Περιγραφή
Η εφαρμογή στοχεύει στη διαχείριση και ανάλυση προϋπολογισμού, 
προσφέροντας βασικές λειτουργίες υπολογισμού εισοδημάτων, εξόδων και ισοζυγίου.

## Ομάδα: 22 (eproipologismos)

- Χρήστος Υποφάντης
- Ειρήνη Τσιγαρίδα
- Μαρία Σκουφέζη 
- Χρήστος Σωτηρίου
- Μανώλης Παπαδημητρίου 
- Ηλίας Μανέτας
